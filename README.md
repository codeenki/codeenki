## 🌊 Hi! Luis over here.

**codeenki** = `code` + **Enki** — the Sumerian deity of water, wisdom, magic, and creation.

That's what drives me as a developer: crafting software with intention, building systems that actually work, and treating every project like something worth creating right. Not just writing code — *creating* with it.

---

### ⚡ Who am I?

I'm a Software Developer with **3 years of experience** building real systems in corporate environments — not just features, **entire systems**. That's the part I love most: taking a complex problem and architecting a full solution from the ground up.

Now I'm expanding beyond corporate. I want to build all kinds of software — web apps, tools, creative projects, things I haven't even thought of yet. This GitHub is where that journey lives.

I believe in clean logic, solid architecture, and code that someone else can actually read six months later.
